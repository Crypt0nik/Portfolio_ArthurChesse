Présentation de l'entreprise

  • Contexte et activité

    Smile 4 All est une clinique dentaire moderne située au centre-ville, spécialisée dans les soins dentaires généraux et esthétiques. Avec une équipe de 6 professionnels, le cabinet accueille plus de 1500 patients par an.

    Fondé il y a 10 ans par Roland Marceau, Smile 4 All s'est développé pour devenir une référence locale en matière de soins dentaires de qualité. L'établissement mise sur les technologies modernes et un accueil personnalisé.

  • Équipe et environnement

    J'ai été intégré dans une équipe dynamique composée de Roland Marceau (chirurgien-dentiste), de 2 assistantes dentaires, 2 secrétaires médicales et d'un gestionnaire administratif. L'environnement de travail était très professionnel et bienveillant.

Missions réalisées

Développement Frontend (TailwindCSS + Vanilla JS)

Création d'un site responsive avec TailwindCSS, développement de Web Components pour les actualités et calendrier, intégration Google Maps, navigation mobile optimisée. Architecture modulaire avec Fetch API.

Backend Node.js + API REST

Serveur Express.js sur port 3001, création d'endpoints REST (/api/news, /api/calendar, /api/auth), gestion uploads images avec Multer, système d'authentification admin, stockage JSON local.

Dashboard Admin + CRUD

Interface d'administration complète avec éditeur Quill.js, CRUD actualités, gestion calendrier consultation/administratif, upload d'images, authentification sécurisée (admin/admin123).
